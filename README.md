# 可鑫的截屏小工具

一个最小可用的桌面截屏工具，目标是接近 QQ 内置截图体验，但只保留截图能力：

- 选区截图快捷键 `CommandOrControl+K`
- 整屏直接复制快捷键 `CommandOrControl+Shift+K`
- 优先截取鼠标所在的当前屏幕
- 全屏截图覆盖层
- 鼠标拖拽选区
- 复制到剪贴板
- 保存为 PNG
- `Esc` 取消，`Enter` 复制，`Cmd/Ctrl + S` 保存

## 运行

```bash
npm install
npm start
```

## macOS 权限

首次使用如果无法抓到画面，需要授予“屏幕录制”权限：

1. 打开“系统设置”
2. 进入“隐私与安全性”
3. 打开“屏幕录制”
4. 允许当前应用，或者允许启动它的终端 / IDE
5. 重新启动应用

## 说明

这个版本只做截屏主流程，没有加入标注、贴图、OCR、历史记录或长截图。
